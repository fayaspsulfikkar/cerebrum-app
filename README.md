# CEREBRUM

Neural Engagement Analysis Platform

## About

Cerebrum analyzes how the human brain responds to video content by measuring activation across 6 neural regions using Meta's TRIBE v2 foundation model.

## Features

- Google OAuth authentication
- Brain region activation analysis (6 regions)
- Timeline heatmaps
- Radar charts
- Side-by-side video comparison
- Category benchmarks
- CSV & PDF export

## Built With

- Flask (Python backend)
- TRIBE v2 (Meta's brain activity prediction model)
- PostgreSQL (database)
- Plotly/Matplotlib (visualizations)
